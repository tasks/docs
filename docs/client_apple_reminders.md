title: Apple Reminders
---

You can synchronize Tasks.org with Apple Reminders using Tasks.org Cloud, DAVx5, or CalDAV.

Tasks.org Cloud users will need to [generate an app password](app_passwords) before getting started.

## macOS

1. Open **System Settings** and navigate to **Internet Accounts**
2. Click **Add Account** and click **choose from a list**
3. Click **Add Other Account** and select **CalDAV Account**
4. Change **Account Type** to **Manual**
5. Enter your server, username, and password
6. Click **Sign In**
7. Make sure **Reminders** are toggled on
    * Tasks.org Cloud Subscribers: Toggle on **Calendars** if you would like to store them in Tasks.org Cloud
    * Tasks.org Cloud Guests: Toggle off **Calendars**

Your Tasks.org lists will appear in Reminders.

## iOS

1. Open **Settings** and navigate to **Apps**
2. Tap on **Reminders**, then tap **Reminder Accounts**
3. Tap **Add Account** and select **Add Other Account...**
4. Tap **Add CalDAV Account**
5. Enter your server, username, and password
6. Tap **Next**
7. Make sure **Reminders** are toggled on and tap **Save**
    * Tasks.org Cloud Subscribers: Toggle on **Calendars** if you would like to store them in Tasks.org Cloud
    * Tasks.org Cloud Guests: Toggle off **Calendars**

Your Tasks.org lists will appear in Reminders.
